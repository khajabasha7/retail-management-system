# 🏪 Retail Management System

A full-stack Retail Management System built with **MERN Stack** that manages products, inventory, employees, sales transactions, and role-based access.

The system supports three roles:

- Admin
- Manager
- Cashier

Each role has different permissions and features.

---



# 📌 Features

## 🔐 Authentication

- User registration
- User login
- JWT based authentication
- Password hashing using bcrypt
- Role-based authorization


## 👨‍💼 Admin Dashboard

Features:

- Create users
- Manage employees
- View employee details
- Role management


## 📦 Manager Dashboard

Features:

### Product Management

- Add products
- Update product details
- Delete products
- Search products
- Pagination support


### Inventory Management

- View stock levels
- Add stock
- Monitor low stock products


### Sales Reports

- Daily sales
- Monthly sales overview


## 🧾 Cashier Dashboard

Features:

### New Sale

- View available products
- Select multiple products
- Add quantity
- Calculate total amount
- Complete sales transaction


### Inventory View

- Check available products
- View current stock
